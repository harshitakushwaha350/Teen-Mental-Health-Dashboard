# Teen-Mental-Health-Dashboard
Power BI dashboard analyzing teen mental health and social media usage.
🧠 Teen Mental Health Dashboard

📊 Overview  
This project is a Power BI dashboard designed to analyze and visualize mental health trends among teenagers.  
It helps in understanding key factors affecting mental well-being such as stress, anxiety, sleep patterns, academic pressure, and lifestyle habits.

🎯 Objective  
- To analyze mental health patterns in teenagers  
- To identify key stress factors (study pressure, sleep, social media usage, etc.)  
- To create interactive visual insights using Power BI  
- To support awareness and early detection of mental health issues  

🛠️ Tools & Technologies  
- Power BI 📊  
- Microsoft Excel 📁  
- Data Visualization 📈  
- Data Cleaning & Transformation  

📌 Key Features  
- Interactive dashboard with filters  
- Mental health trend analysis  
- Stress vs lifestyle comparison  
- Sleep vs academic pressure insights  
- Easy-to-understand visual charts
